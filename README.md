## 📚 Sobre o projeto
Este projeto tem como objetivo aplicar conceitos e ferramentas de Ciência de Dados, incluindo etapas de:
- coleta e preparação de dados;
- análise exploratória;
- visualização de dados;
- modelagem (quando aplicável);
- interpretação dos resultados.

## 🎯 Objetivos
- Praticar o uso de ferramentas da disciplina;
- Desenvolver um fluxo de trabalho em Ciência de Dados;
- Gerar insights a partir de dados reais/simulados;
